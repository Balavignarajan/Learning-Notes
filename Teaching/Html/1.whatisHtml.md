## What is HTML?

- HTML stands for Hyper Text Markup Language.
- HTML is the standard markup language for creating Web pages.
- HTML describes the structure of a Web page.
- HTML elements tell the browser how to display the content.

**-----------------------------------------------------------------------------------------------------------**

## A Simple HTML Document

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>
</html> 
      
    - Example Explained -
        - 
           - <!DOCTYPE html> → Tells the browser “This is an HTML5 page. 
           - The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly.
           - It must only appear once, at the top of the page (before any HTML tags).
        - The <html> element is the root element of an HTML page
        - <head> → Hidden part (title, links, metadata).
        - The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).
        - <body> → Visible part (what users actually see).such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
        - The <h1> element defines a large heading.
        - The <p> element defines a paragraph.

**-----------------------------------------------------------------------------------------------------------**

## HTML Headings

- HTML headings are defined with the <h1> to <h6> tags.
- <h1> defines the most important heading. <h6> defines the least important heading: 
     - <h1>This is heading 1</h1>
     - <h2>This is heading 2</h2>
     - <h3>This is heading 3</h3>
     - etc h1 to h6

**-----------------------------------------------------------------------------------------------------------**

## HTML Paragraphs

- HTML paragraphs are defined with the <p> tag:
  -Example : - <p>This is a paragraph.</p> - <p>This is another paragraph.</p>

**-----------------------------------------------------------------------------------------------------------**

## HTML Links

     - The <a> tag is used to create links (anchors) in HTML. The main purpose is to navigate to a different page or a specific part of the current page when the link is clicked.
     - HTML links are defined with the <a> tag. The link address is specified in the href attribute:
         - *Example* :
                - <a href="https://www.example.com">This is a link</a>
                - <a href="https://www.google.com">Google</a>
                - <a href="about.html">About Us</a>

      - <a> — This is the anchor tag itself.
      - href="https://www.example.com" — This is the most important attribute of the <a> tag. It stands for Hypertext Reference, and it tells the browser where the link should go when it’s clicked. The href attribute can contain:
      - A URL (like https://www.example.com).
      - A path to another page in your website (like about.html or /contact).

**-----------------------------------------------------------------------------------------------------------**

## HTML Images

- The <img> tag is used to embed images in an HTML page. It is a self-closing tag and does not require a closing tag.
- The source file (src), alternative text (alt), width, and height are provided as attributes:
  - _<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">_

**-----------------------------------------------------------------------------------------------------------**

## HTML Attributes

- All HTML elements can have attributes
- The href attribute of <a> specifies the URL of the page the link goes to
- The src attribute of <img> specifies the path to the image to be displayed
- The width and height attributes of <img> provide size information for images
- The alt attribute of <img> provides an alternate text for an image
- The style attribute is used to add styles to an element, such as color, font, size, and more.

## Division & Span

🧱 What Are <div> and <span>?

     - Both <div> and <span> are called “container tags” in HTML.

🎯 WHY We Need Them?

     - Imagine you are building a house (a webpage):
     - You have walls, rooms, furniture, etc.
     - You need to divide the house into sections (like bedroom, kitchen, hall) to keep it organized.
     👉 In HTML, the <div> and <span> tags help you divide your web page into sections or parts, so you can style or control them separately using CSS later.

💡 div = block element
💡 span = inline element - <div>This is a block-level container.</div> - <span>This is inline text.</span>

_Div_

✅ Example 2 — Adding Styles to Div

  <div style="background-color: lightblue; padding: 10px; margin: 10px;">
    <h2>Welcome to My Website</h2>
    <p>This section has a blue background!</p>
  </div>

  <div style="background-color: lightgreen; padding: 10px; margin: 10px;">
    <h2>Contact Me</h2>
    <p>Email: bala@example.com</p>
  </div>

🧠 Remember:

_<div> always starts on a new line (it’s a block element)._

- It’s like a big box around a group of content.

_Span_

- ✨ <span> → Inline Tag (Small container)
- Think of <span> as:

  - A small container used to style or highlight part of a text inside a line.
  - Unlike <div>, it does not break the line — it stays inline with the text.

    ✅ Example 1 — Highlighting Words

      <p>Hello, my name is <span style="color: red;">Bala</span> and I love.</p>

_Diffrence between span and div_

- Feature <div> <span>

1. Type - Block-level - Inline
2. Use - To group big sections - To style small parts of text
3. Line Break - Starts on a new line -Stays in same line

🧠 Easy Memory Tip:
👉 <div> → “Divide the page into boxes”
👉 <span> → “Span a few words only”

## Html Tables

🧾 What is a Table?

- A table in HTML is just like a table in your notebook or Excel sheet —
  it shows data in rows and columns.

- 📖 Example from real life:
  | Name | Age | City |
  | ----- | --- | ------- |
  | Bala | 21 | Chennai |
  | Kiran | 22 | Madurai |

🧩 Basic Table Tags

        | Tag       | Meaning      | Description                              |
     | --------- | ------------ | ---------------------------------------- |
     | `<table>` | Table        | The main container that holds everything |
     | `<tr>`    | Table Row    | Creates a row (horizontal line)          |
     | `<th>`    | Table Header | Creates heading cells (bold + centered)  |
     | `<td>`    | Table Data   | Creates normal data cells                |

🧱 Basic Table Example

  <table border="1">
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>City</th>
    </tr>
    <tr>
      <td>Bala</td>
      <td>21</td>
      <td>Chennai</td>
    </tr>
    <tr>
      <td>Kiran</td>
      <td>22</td>
      <td>Madurai</td>
    </tr>
  </table>

🧩 Let’s break it down slowly:

1️⃣ <table border="1">
→ This creates a table and adds a border so you can see the boxes clearly.
(Without border, it’s invisible!)

2️⃣ <tr> means Table Row
→ Each row is like a horizontal line in the table. (row and horizontal are side by side).

3️⃣ <th> means Table Header
→ These are the titles or labels (Name, Age, City).
→ They appear bold and centered by default.

4️⃣ <td> means Table Data
→ These are the normal cells that contain your actual data.

_Adding More Style (Attributes)_

- Let’s make the table look neater by adding some extra attributes:

      <table border="1" cellpadding="10" cellspacing="5" align="center" width="60%">

     <tr>
       <th>Product</th>
       <th>Price</th>
       <th>Quantity</th>
     </tr>
     <tr>
       <td>Apple</td>
       <td>₹50</td>
       <td>10</td>
     </tr>
     <tr>
       <td>Banana</td>
       <td>₹30</td>
       <td>12</td>
     </tr>
   </table>

  ✨ What these new attributes mean:

              | Attribute     | Meaning                       | Example            |
      | ------------- | ----------------------------- | ------------------ |
      | `border`      | Adds border around cells      | `border="1"`       |
      | `cellpadding` | Space inside each cell        | `cellpadding="10"` |
      | `cellspacing` | Space between each cell       | `cellspacing="5"`  |
      | `align`       | Centers the table             | `align="center"`   |
      | `width`       | Makes table wider or narrower | `width="60%"`      |

🧩 Table with Header, Body, and Footer (for clarity)
As you build bigger tables, it’s good to divide them into parts:

  <table border="1" cellpadding="8" width="70%">
    <thead>
      <tr>
        <th>Subject</th>
        <th>Marks</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Math</td>
        <td>95</td>
      </tr>
      <tr>
        <td>Science</td>
        <td>88</td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <th>Total</th>
        <th>183</th>
      </tr>
    </tfoot>
</table>

🧠 Think of:

<thead> → Table Head (top)
<tbody> → Table Body (middle)
<tfoot> → Table Foot (bottom)

🧠 Quick Summary (like a cheat note)

| Tag                                    | Purpose               |
| -------------------------------------- | --------------------- |
| `<table>`                              | Creates the table     |
| `<tr>`                                 | Creates a new row     |
| `<th>`                                 | Creates header cells  |
| `<td>`                                 | Creates data cells    |
| `<thead>`, `<tbody>`, `<tfoot>`        | Organize large tables |
| `border`, `cellpadding`, `cellspacing` | Make table readable   |

## HTML FORMS

🧾 What is a Form?

👉 In real life:
When you visit a website like Facebook or Gmail, you see boxes to type your name, email, or password — that’s a form.

👉 In HTML:
A form is a way to collect information from the user — for example:

- Login page (Username + Password)
- Registration page (Name, Email, Password)
- Search box (Search term)
- Feedback form (Message)

🧩 The Main Form Tag

<form>
  <!-- form elements go here -->
</form>

- Everything that collects user input (like text boxes, buttons, etc.) must be placed inside the <form> tag.

✍️ Common Form Elements

| Tag          | Purpose                               | Example                                 |
| ------------ | ------------------------------------- | --------------------------------------- |
| `<input>`    | For user input (text, password, etc.) | `<input type="text">`                   |
| `<label>`    | Label for input (shows field name)    | `<label>Name:</label>`                  |
| `<textarea>` | For longer text (like comments)       | `<textarea></textarea>`                 |
| `<select>`   | Dropdown list                         | `<select><option>...</option></select>` |
| `<button>`   | For submitting or resetting form      | `<button>Submit</button>`               |

---------------------------------------------------------------------------------------------------------

🧱 Simple Example – Login Form

<form>
  <label>Username:</label>
  <input type="text"><br><br>

<label>Password:</label>
<input type="password"><br><br>

<button type="submit">Login</button>

</form>

🧩 Let’s understand it step by step
1️⃣ <form>
→ It’s the container that holds everything.0
2️⃣ <label>
→ It tells the user what to type. Example: “Username:” or “Password:”
3️⃣ <input type="text">
→ Creates a text box.
4️⃣ <input type="password">
→ Hides the text (shows dots ••••) when typing.
5️⃣ <br>
→ Adds line breaks (moves to the next line).
6️⃣ <button type="submit">
→ Adds a submit button.

## Input Types in Html

🧩 What is an Input Type?

- In a form, we use the <input> tag to get information from the user.
But the type of input decides what kind of data you want — text, number, password, date, checkbox, etc.
💡 Example:
<input type="text">
<input type="password">
Here, both are <input> tags — but they behave differently because of their type.

🧱 Most Common HTML Input Types (With Examples)

1️⃣ Text Input
Used for typing plain text — like name or username.
<label>Name:</label>
<input type="text" placeholder="Enter your name">

2️⃣ Password Input
Used for passwords — hides the text with dots (••••).
<label>Password:</label>
<input type="password" placeholder="Enter your password">

3️⃣ Email Input
Used to collect email addresses.
Browser will warn you if it’s not in the right format (example: abc@gmail.com).
<label>Email:</label>
<input type="email" placeholder="Enter your email">

4️⃣ Number Input
Used to type numbers only (not text).
<label>Age:</label>
<input type="number" min="1" max="100">

5️⃣ Date Input
Used to select a date from a calendar
<label>Birthday:</label>
<input type="date">

6️⃣ Radio Buttons
Used when you want to choose only ONE option from a group.
<label>Gender:</label><br>
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

7️⃣ Checkboxes
Used when you want to choose multiple options.
<label>Hobbies:</label><br>
<input type="checkbox" value="reading"> Reading
<input type="checkbox" value="music"> Music
<input type="checkbox" value="traveling"> Traveling

8️⃣ File Input
Used to upload files (like photos, PDFs, etc.)
<label>Upload your photo:</label>
<input type="file">

9️⃣ Color Picker
Lets the user pick a color 🎨
<label>Pick your favorite color:</label>
<input type="color">

🔟 Range Slider
Lets the user select a number using a sliding bar.
<label>Volume:</label>
<input type="range" min="0" max="100">

1️⃣2️⃣ Submit Button
Used to send the form data.
<input type="submit" value="Submit Form">


1️⃣3️⃣ Reset Button
Used to clear all fields in the form.
<input type="reset" value="Clear All">

🧠 Bonus Tip — Input Attributes You’ll Use Often
<form>
  <label>Name:</label>
  <input type="text" placeholder="Enter your name"><br><br>

  <label>Username:</label>
  <input type="text" value="Bala"><br><br>

  <label>Email:</label>
  <input type="email" placeholder="Enter your email" required><br><br>

  <label>Country:</label>
  <input type="text" value="India" readonly><br><br>

  <label>Phone:</label>
  <input type="text" value="9876543210" disabled><br><br>

  <input type="submit" value="Submit">
</form>

steps to instuction :
🧩 1️⃣ placeholder — Shows hint text inside the input box
🧩 2️⃣ value — Sets a default value - ✨ The box will already show “Bala” when the page loads.
🧩 3️⃣ required — Makes field mandatory - ✨ If you click submit without typing, it shows a warning 🚫
🧩 4️⃣ readonly — Can’t edit, only view - User can see the value but can’t change it.
🧩 5️⃣ disabled — Makes input inactive - Gray and unclickable — user can’t type or select.

## <video> / <audio> <iframe>

*1. Video Tag*

🎥 What is the <video> Tag?

The <video> tag is used to show videos on a web page —
just like how YouTube plays videos, but in your own website!

It allows you to play, pause, and control videos directly in the browser.

🧱 Basic Example:

<video width="400" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

🧩 Let’s break it down slowly:

Got it! Here’s the information without the table format:

* **`<video>`**: The main tag that displays video.
  Example: `<video>...</video>`

* **`width="400"`**: Sets the width of the video (in pixels).
  Example: 400 = medium size.

* **`controls`**: Adds play, pause, and volume buttons to the video.
  Example: `<video controls>`

* **`<source>`**: Tells the browser where to find the video file.
  Example: `<source src="movie.mp4" type="video/mp4">`

* **`"Your browser does not support..."`**: This message appears if the browser can’t play the video.
  Example: Fallback text like "Your browser does not support the video tag."


🎛️ Optional Video Attributes :

You can add extra features to the <video> tag:
| Attribute  | Meaning                            | Example                          |
| ---------- | ---------------------------------- | -------------------------------- |
| `autoplay` | Starts playing automatically       | `<video autoplay>`               |
| `loop`     | Repeats video forever              | `<video loop>`                   |
| `muted`    | Starts with no sound               | `<video muted>`                  |
| `poster`   | Shows an image before video starts | `<video poster="thumbnail.jpg">` |

💡 Example:
<video width="400" controls autoplay muted loop poster="preview.jpg">
  <source src="nature.mp4" type="video/mp4">
</video>

*2. Audio Tag*

 🎵 What is the <audio> Tag?

Just like <video>, the <audio> tag is used to play sound or music on your webpage 🎶

🧱 Basic Example:

  Your browser does not support the audio element.
</audio>

🧩 Breakdown:
| Tag / Attribute | Meaning                               | Example                                      |
| --------------- | ------------------------------------- | -------------------------------------------- |
| `<audio>`       | The main tag for playing audio        | `<audio>...</audio>`                         |
| `controls`      | Adds play/pause buttons               | `<audio controls>`                           |
| `<source>`      | Tells browser the music file and type | `<source src="music.mp3" type="audio/mpeg">` |

🎛️ Optional Audio Attributes
| Attribute  | Meaning                       | Example            |
| ---------- | ----------------------------- | ------------------ |
| `autoplay` | Starts the song automatically | `<audio autoplay>` |

| `muted`    | Starts silent                 | `<audio muted>`    |

💡 Example:
<audio controls autoplay loop>
  <source src="background.mp3" type="audio/mpeg">
</audio>
